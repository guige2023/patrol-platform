## 🔴 最严重的3个问题

### 1️⃣ 权限检查异步实现错误 (C1)
**威胁等级**: 🔴 CRITICAL  
**后果**: 权限检查失效，任何用户可能获得不应有的权限  
**修复时间**: 4小时  
**实施复杂性**: ⭐ 简单

```python
# 问题代码
async def check_permission(...):
    db = AsyncSessionLocal()  # 错误：同步创建
    await db.execute(...)     # 错误：异步执行
    
# 正确方式
async def check_permission(..., db: AsyncSession = Depends(get_db)):
    await db.execute(...)
```

---

### 2️⃣ 路径遍历漏洞 (C5)
**威胁等级**: 🔴 CRITICAL  
**后果**: 攻击者可读取任意文件（如 /etc/passwd）  
**修复时间**: 4小时  
**实施复杂性**: ⭐ 简单

```python
# 问题代码
@router.get("/{knowledge_id}/{filename}/preview")
async def preview(filename: str):  # 无验证！
    file_path = f"/uploads/{filename}"  # ../../../etc/passwd

# 正确方式
async def preview(file_id: UUID):  # 使用ID而非filename
    attachment = await db.get(Attachment, file_id)
```

---

### 3️⃣ CSRF保护缺失 (C2)
**威胁等级**: 🔴 CRITICAL  
**后果**: 跨站请求伪造，恶意网站可代表用户执行操作  
**修复时间**: 6小时  
**实施复杂性**: ⭐⭐ 中等

```python
# 添加CSRF中间件
from fastapi_csrf_protect import CsrfProtect

@router.post("/users")
@CsrfProtect.protected()
async def create_user(...):
    pass
```
